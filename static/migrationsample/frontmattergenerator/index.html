Front Matter Generator

Use this tool to generate consistent front matter for your MS Word documents (web pages) and tagging for your submitted PDFs and images.

Use this tool to create the front matter for each of your MS Word web pages and to include appropriate metadata with every PDF/document that's submitted on AAP.org/websites:

  • Complete the form
  • Select "Generate Front Matter"
  • Copy the front matter from the blue box and
    • New Web Pages: Paste it to the first line of your Word doc (.docx)
    • New PDFs or Images to AAP.org: Paste it into the "Comments" box of the request form you're already used to using on AAP.org/websites. The "Title" tag for PDFs should be the title of the PDF (i.e., what you see at the top when opening the PDF) and not the web page you're posting it to.
  • If you're submitting a new page or new PDF that is gated (i.e., for members only), please let us know in your request.

This form works best with the most recent versions of FireFox and Chrome or Internet Explorer 11. Internet Explorer 9 is not supported.

The AAP.org tag search provides more information on the available tags.
